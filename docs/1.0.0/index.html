Descrição

O componente gumga-menu-ng gera um menu lateral baseado em 2 arquivos keys-url e menu-url, a partir do itens do menu, é verificada se existe a permissão em keys e então, é apresentado no menu.

Pré requisitos

  • Gumga Translate

Instalação

// npm
npm i gumga-menu-ng --save

// bower
bower install gumga-menu-ng --save
Usando
  • HTML
  • JavaScript
<body gumga-translate="pt-br">
  <gumga-menu menu-url="gumga-menu.json" keys-url="keys.json" image="./gumga.png"></gumga-menu>
<body>
